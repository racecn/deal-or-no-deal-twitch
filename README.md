# Deal or No Deal Game Show Control System

## Setup Instructions

### Backend Setup
1. cd backend
2. python -m venv venv
3. source venv/bin/activate  # On Windows: .\venv\Scripts\activate
4. pip install -r requirements.txt
5. python app.py

### Frontend Setup
1. cd frontend
2. npm install
3. npm start

## Usage
- Admin interface: http://localhost:3000/admin
- Player display: http://localhost:3000
